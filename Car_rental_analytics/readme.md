🚗 Car Rental Analytics & Profit Prediction System

An end-to-end data analytics and machine learning project that analyzes car rental business performance and predicts contract profitability.
The system integrates PostgreSQL, Python, Pandas, Machine Learning, and Streamlit to transform raw transactional data into actionable insights through an interactive dashboard.

📊 Project Overview

This project focuses on analyzing rental contract performance and predicting profitability using data analytics and machine learning techniques.

The pipeline includes:

Data modeling and storage in PostgreSQL

Data processing and KPI analytics using Python & Pandas

Machine learning models to predict rental contract profit

Interactive Streamlit dashboard for business insights and visualization

The goal is to help businesses identify profitable segments, optimize fleet strategy, and improve financial performance.
